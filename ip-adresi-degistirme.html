IP Adresi Değiştirme

Static IP adresi kendi kendine değişmeyen durağan bir ip çeşididir. Herhangi bir sebepten ötürü değiştirilmek istenmesi durumunda internet sağlayıcısıyla irtibata geçilmesi ve static ip adresinin değiştirilmesi için talepte bulunulması gerekir. Static IP adresi, dinamic IP adresi gibi modemin kapatılıp açılması ile değişmemektedir.

Dinamic IP adresi ise değişken bir ip çeşidi olup modemin kapatılıp açılması ile kolayca değiştirilebilir. Bunun için modeminizi kapama tuşundan kapatınız. Eğer bir kapama tuşu yoksa elektrik bağlantısını kesiniz. Yaklaşık 10 saniye kadar bekledikten sonra modeminizi tekrar açınız. Bu işlemin ardından Dinamic IP adresiniz otomatik olarak değişecektir. IP adresinizin değişip değişmediğinin kontrolünü, ip adresim ne sayfasından yapabilirsiniz. (ip numaranızın değişip değişmediğini kontrol ederken, modemi kapatıp açtıktan sonra ip adresinizin göründüğü sayfayı (kapatmış ve yeniden açmış olsanız dahi) tekrardan yenilemeniz gerekmektedir. Aksi durumda, ip numaranız değişmiş olsa bile tarayıcıların caching özelliğinden ötürü sayfa üzerinde eski ip adresiniz görünmeye devam edecektir.)

Aynı zamanda bu işlem, ne tür bir ip adresine sahip olduğunuzu da belirlemenize imkan tanıyacaktır. Şayet ip adresiniz yukarıda belirtilen işlemin ardından otomatik olarak değişmiyorsa bu durum, ip adresinizin static yani durağan bir ip adresi olduğu anlamına gelecektir.
