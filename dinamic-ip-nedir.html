Dinamic IP Adresi Nedir ?

Dinamic IP adresi, değişken veya değiştirilebilen IP adresi demektir. Özel olarak static IP başvurusu yapılmadıysa ISS tarafından satın alacağınız internet hizmetleri dinamic ip ile gelir. Servis sağlayıcıları tarafından dinamic IP adresi için herhangi bir ek ücret talep edilmez. Modemin kapatılıp tekrardan açılması ile dinamic IP adresi kolayca değiştirilebilir. Şayet dinamic IP adresiniz ile web platformunda problem yaşıyorsanız, modemi kapatıp açarak dinamic IP nizi kolayca değiştirebilir ve söz konusu problemleri giderebilirsiniz. Detaylı bilgi için ip adresi değiştirme sayfasını ziyaret ediniz.

Hosting vs. gibi işlemlerin dinamic IP adresi kullanılarak yapılması, en ufak bir elektrik kesintisinde veya modemin kapatılıp açılmasında hizmetin aksaklığa uğramasına sebep olacaktır. Çünkü hizmetlerinizi yönlendirdiğiniz IP adresi değişmiş olacak ve bu da sistemde tanımlı olan IP adresinin bulunamayarak hizmetin aksamasına neden olacaktır. Bu sebepten ötürü bu tür işlemlerin Static IP adresi üzerinden yapılması gerekir.

Fakat No-ip veya DyDns gibi hizmetlerden yararlanılarak dinamic IP adresi ile söz konusu hosting vs. gibi hizmetler sağlanılabilir.

Normal bilgisayar ve internet kullanıcıları için uygun olan ip modeli dinamic IP dir.

IP adresinizin dinamic IP olup olmadığını öğrenmek için ip adresi sorgulama sayfasından ip numaranızı öğrenin. Devamında modeminizi kapatarak 10 saniye bekledikten sonra tekrar açın. Önceden girdiğiniz ip sorgulama sayfasına tekrar girin ve tarayıcıdan sayfayı yenileyin. (Tarayıcıdan sayfayı yenilemeniz gereklidir. Aksi durumda sayfayı kapatıp tekrar açmış olsanız bile caching özelliğinden ötürü eski ip adresiniz görünecektir.) Şayet ip numarası değişiyorsa dinamic IP adresine sahipsiniz demektir. Eğer ip adresiniz değişmiyorsa static IP adresine sahipsiniz demektir.
