Static IP Adresi Nedir ?

Static IP adresi, adından da anlaşılacağı gibi durağan IP yani değişmeyen IP adresidir. ISS tarafından internet paketi alırken veya aldıktan sonra telefon ile static IP başvurusu yapılabilmektedir.

ISS tarafından atanan static ip adresi sadece size özel olup, her ay internet paket tarifenize eklenmek suretiyle 2 veya 3 tl gibi bir rakamla ücretlendirilir.

Static IP adresi sabit olup dinamic IP adresi gibi değişken değildir ve modemi açıp kapatma yöntemiyle değişmez. Static IP adresi'nin değiştirilmesi için servis sağlayıcısıyla irtibata geçilmesi ve yeni bir static IP talebinde bulunulması gerekir. Detaylı bilgi için ip adresi değiştirme sayfasını ziyaret ediniz.

Static IP adresi genellikle; Hosting, Mail sunucuları, VPN, RDP bağlantıları gibi hizmetler için tercih edilir. Örneğin, bir kullanıcı kendi bünyesinde bir hosting oluşturmak ve bu hosting üzerinden bir web sitesi yayını yapmak istiyorsa, static bir ip adresi alması ve web sitesini yayınlayacağı domain adresini, bu ip adresine yönlendirmesi gerekir. Çünkü hosting ve web sitesi işlemleri için IP adresinin sabit olması gerekir. Burada domain yönlendirmesi ip adresi için isim ataması demektir. Aksi durumda IP adresinin değişmesi, domainin yönlendiği IP adresinin erişilememesine ve bu yayının durmasına neden olacaktır.

IP adresinizin static IP olup olmadığını öğrenmek için ip adresim sayfasından ip numaranızı öğrenin. Devamında modeminizi kapatarak 10 saniye bekledikten sonra tekrar açın. Önceden girdiğiniz ip no bul sayfasına tekrar girin ve tarayıcıdan sayfayı yenileyin. (Tarayıcıdan sayfayı yenilemeniz gereklidir. Aksi durumda sayfayı kapatıp tekrar açmış olsanız bile caching özelliğinden ötürü eski ip adresiniz görünecektir.) Şayet ip numarası değişmiyorsa static IP adresine, değişiyor ise dinamic IP adresine sahipsiniz demektir.
